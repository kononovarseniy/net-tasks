С телефона подключится к ar-net:48785. Соединение должно пройти успешно.
Проверка подключения завершена.

Настройка iptables:
1) Не убивать ответы сайтов:
    sudo iptables -A INPUT -m conntrack --cstate RELATED,ESTABLISHED -j ACCEPT
2) Запрет подклбчения ко всем портам кроме 22(ssh) через все интерфейсы кроме loopback:
    sudo iptables -A INPUT ! -i lo -p tcp ! --dport 22 -j DROP
Обе строки содержатся в скрипте set-rules.sh
Откат изменений:
    sudo iptables -F INPUT
    Либо скриптом reset-rules.sh

Теперь соединение описанным выше способом не удастся.
